# inyaa-sakura-tailwind

inyaa-sakura-daisy是inyaa系列的博客前端采用了nuxt3，vue3，ts等技术开发的博客，在样式上参考了其他作品，之前有做过类似sakura那种，看腻了，就换了成了这种大图的版本。


# 技术组成

项目整体上来说，还在开发中，并且提前声明，我不是专业前端，我本职是java，前端只是我的个人爱好。
所以很多代码风格，可能会很怪

* [x] `nuxt3`
* [x] `vue3`
* [x] `typescript`
* [x] `windicss`
* [x] `dayjs`
* [x] `markdown-it`
* [x] `highlight`
* [x] `daisyui`
